# Stratégie Go-to-Market - Affinity Mapping Tool

**Skripta** est un outil tout-en-un pour la recherche utilisateur:

| Feature | Description |
|---------|-------------|
| **Transcription** | Convertit audio/video en texte (AI) |
| **Extraction insights** | Manuel ou automatique (AI) |

## 💡 Présentation École - Conseils

### Structure Recommandée (10-15 min)

1. **Problème (2 min)** - Montre la difficulté actuelle
   - "Combien de temps passez-vous à grouper des post-its?"
   - Stats: 3-5h par session d'affinity mapping

2. **Solution (3 min)** - Démo live
   - Crée un cluster en 1 clic
   - Génère un persona
   - Montre le dot voting

3. **Différenciation (2 min)** - Pourquoi c'est unique
   - IA française (Groq)
   - Collaboration temps réel
   - Export ready pour presentations

4. **Business Model (2 min)** - Comment ça marche
   - Tiers de prix
   - Comment on gagne de l'argent

5. **Demo (5 min)** - Le produit en action

### Tips pour les étudiants

- **Sois authentique** - Ils préfèrent un founder sincère qu'un présentateur parfait
- **Montre les coulisses** - Les bugs, les échecs, ça les rassure
- **Demande du feedback** - "Qu'est-ce qui vous embêterait dans cet outil?"
- **Montre que tu écoutes** - Prends des notes de leurs retours

## 🎓 Crédits - Système Proposé

### Comment ça marche

```
FREE:     50 insights   → 100 jetons
STARTER: 500 insights  → 1,000 jetons  
PRO:     Illimité     → Illimité
```

### Dépense des jetons

| Action | Jetons |
|--------|--------|
| Générer un cluster AI | 10 jetons |
| Générer un persona | 15 jetons |
| Rename automatique (IA) | 5 jetons |
| Thematic analysis | 20 jetons |

### Pourquoi ce système

1. **Freemium attractif** - 50 insights gratuits, assez pour tester
2. **Pas de surprise** - L'utilisateur sait ce que ça coûte
3. **Scalable** - Les gros utilisateurs paie plus
4. **Psychologie** - Le zero-price effect fait que les 50 premiers insights sont "gratuits"

### Alternative: Abonnement Simple

```
Free: 50 insights max
Paid: Tout illimité pour €19-35/mois
```

**Recommandation:** Commencer avec l'abonnement simple, ajouter les jetons plus tard si besoin.
